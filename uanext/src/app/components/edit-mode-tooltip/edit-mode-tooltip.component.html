Чтоб разместить 3Д объект на карте нужно:
Выбрать проект (в правой части верхнего меню), если еще нету ни одного проекта, то создать (нажав в меню кнопку "проекты")
Перетащить понравившийся 3Д объект на карту
Выбрать нужный 3Д объект, кликнув на нем
Перемещать объект клавишами A, S, D, W, или зажав клавишу X, и перемещая мышку
Поворачивать объект клавишами Z и C
Увеличить/уменьшить объект клавишами Z, C
